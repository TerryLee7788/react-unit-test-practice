# React 單元測試練習

## 請先 clone 此專案
```
git clone https://github.com/TerryLee7788/react-unit-test-practice.git
```

## 安裝相依套件
```
npm i
```

```
npm test
```

## 查看涵蓋率
```
npm run test:coverage
```
